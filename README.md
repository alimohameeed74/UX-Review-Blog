# UX-Review-Blog

A modern web interface inspired by the **Neubrutalism** design trend — a mix of **bold shapes**, **vibrant colors**, and **strong visual contrast**.  
This project focuses on creating a **trendy, eye-catching UI** while applying advanced **CSS techniques** in a practical and creative way.

## Overview

This project demonstrates how **Neubrutalism** can be applied in modern web design to achieve a **bold yet functional** user interface.  
Every element — from the buttons to layout — is styled to reflect the playful, geometric essence of Neubrutalism while maintaining usability and responsiveness.

---

## Features

- **Neubrutalism Design:** Strong outlines, vibrant backgrounds, and geometric elements.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens.
- **Interactive Elements:** Hover and click animations using CSS transitions.
- **CSS Techniques:** Utilizes Flexbox, Grid, Animations, Pseudo-classes, and CSS Variables.
- **Clean Code:** Well-organized HTML and CSS structure for easy maintenance.

---

## Technologies Used

- **HTML5**
- **CSS3** (Flexbox, Transitions, Variables)
